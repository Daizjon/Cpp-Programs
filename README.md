# C++ CLI Toolkit

A collection of C++ command-line programs demonstrating systems programming, file parsing, data structures, algorithms, and interactive terminal applications.

All projects were developed and tested in Linux environments and compiled with `g++`.



## 📁 Projects Included

### 🔢 Bit-Matrix
Matrix operations using bit-level manipulation.

### 🧮 Bitset-CLI
Bitset operations and command-line utilities.

### ♠ Black-Jack
Terminal-based Blackjack game implementation.

### 🏁 Checkerboard
Pattern generation and formatted grid output.

### ➗ Command-Line Calculator
String parsing and arithmetic expression evaluation.

### 🔐 Cryptography Scheme
Binary XOR-based encryption/decryption tool.

### 🧠 Disassembler
Low-level instruction parsing and formatting.

### 📊 Encode-Decode-Min-Max
Statistical and transformation utilities.

### 🌙 Moonglow
File parsing and structured input evaluation.

### 🖼 PGM-Manipulation
Image processing tools for PGM files:
- `pgminfo`
- `bigwhite`
- `neg`
- `hflip`

### 🔎 Prime-Number-Primer
Prime checking utilities:
- Functional approach
- Functor-based class implementation

### 🏰 Text-Based-MUD
Graph-based text adventure engine with dynamic room parsing.

### ❌ Tic-Tac-Toe
Interactive NxN terminal-based Tic-Tac-Toe game.


## 🛠 Build

Most programs can be compiled with:

```bash
g++ -std=c++17 -O2 -Wall -Wextra file.cpp -o program
```

Some folders include a `Makefile` for convenience:

```bash
make
```
